linux常用指令

发表于2023-04-20|更新于2023-09-09|技术
|阅读量:

备注是以我所理解的含义进行转述，可能并不精确，仅供参考。

操作文件

命令 用法 备注
pwd 打印工作目录 描述当前位置。
ls 列出目录内容 list，描述当前位置所包含的内容。
cd 更改目录 移动所在位置。
touch 创建空文件 仙人抚我顶，结发受长生。
cat 打印文件内容 可能猫猫也很好奇。
vim 使用vim文本编辑器打开文件 i：进入编辑状态；
ESC键：退出编辑状态；
Shift键+:键：操作vim文本编辑器；
w：写入编辑（即保存）；
q：退出vim编辑器；
!：（将某命令）强制执行。
mkdir 创建目录 Linux中的『目录』可以简单类比为Windows系统中的『文件夹』
cp 复制文件和目录 copy
rm 删除文件和目录 suda rm -rf /* 的意思是删除电脑内的所有文件，在虚拟机中执行此操作后需要在任务管理器中中止虚拟器进程，才能顺利地使虚拟机关机。不建议使用。
mv 移动（重命名）文件或目录

设置网络

命令 用法 备注
ifconfig 查看IP地址
ping 测试网络连接 “咚咚咚……”
firewalld 防火墙

控制电脑

命令 用法 备注
reboot 重启
shutdown 关机
sudo 以超级管理员的身份执行命令 需要输入密码

用户管理

命令 用法 备注
useradd 添加用户
usered 删除用户 『usered -r 用户名』
连同用户的主目录一同删除
usermod 修改用户
chmod 修改用户的权限
