用JavaScript求1000以内的质数

发表于2023-05-27|更新于2023-09-10
|阅读量:

代码原理：

  • 检查某个数与比它小的所有质数取余后是否为0，若不是，则加入质数表，然后检查下一个数。

涉及到的知识点：

  1. 数组
    • 创建：group = []
    • 形式：[num1, num2, num3]
    • 追加：group.push[num4]
    • 长度：group.lenth == 4
    • 提取：group[2] == num3
  2. 函数
    • 求平方根：Math.sqrt(25) == 5
  3. 循环
    • for循环：for(起始值; 循环条件; 循环节) {循环内容}
    • 跳出循环：break
